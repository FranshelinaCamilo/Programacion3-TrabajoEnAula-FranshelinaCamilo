# Programacion3-TrabajoEnAula-FranshelinaCamilo

Trabajo en aula - ArrayList de Visitantes

Estudiante: Franshelina Camilo Cabrera

Matrícula: 1000-5493

Materia: Programación 3

Descripción:

Programa desarrollado en Java que permita el registro de 5 visitantes haciendo uso de un arreglo. El programa debe de solicitar por teclado los siguientes datos de los visitantes: nombre, edad y cédula. Luego de haber registrado los 5 visitantes, el programa debe mostrar por pantalla la información que fue almacenada.

Requisitos:

* Utilizar arreglos (luego se nos indicó que debíamos cambiarlos por ArrayList)
* Utilizar ciclos for
* Utilizar entrada de datos con Scanner
* Mostrar los datos organizados en consola
